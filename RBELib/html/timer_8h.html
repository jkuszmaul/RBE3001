void initTimer ( int  timer,
int  mode,
unsigned long  comp

Initializes the specified timer in the specified mode. This header file provides constants for NORMAL operation mode and CTC operation mode, however there are many more modes of operation for the Timers that can be experimented with.

Parameters
timerThe number of the timer to be initialized (0-2).
mode0 if normal, 2 if CTC mode. No other modes currently supported.
compOnly used in CTC mode. The period, in microseconds, based on the clock and prescale.

void setPWM ( unsigned char  timer,
unsigned int  top,
unsigned int  comp 
)

Set the PWM on a given timer. For a given timer, this will cause the timer to count up to TOP and then back down again. When counting up, the pin output OCnB (n=timer) will be set high until the timer compares equal to comp, at which point it will be toggled low and then toggled high again when counting down. The PWM signal itself is output on the OCnB pin.

Parameters
timerThe timer to set the PWM for.
topThe timer value to count up (TOP) before counting down again.
compThe value at which to toggle OCnB.
